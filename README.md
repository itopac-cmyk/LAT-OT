# LLM-powered Advisory Triage for OT-Security (LAT-OT)

## Project Description
I am developing an open-source framework designed to automate the triage of security advisories for Operational Technology (OT) operators. In an era where "Zero-Day" vulnerabilities are becoming the norm and exploit windows are collapsing, manual assessment is no longer feasible. My project leverages Large Language Models to analyze incoming security advisories, correlate them with specific industrial asset inventories, and prioritize mitigation steps. The goal is to provide a tool that helps critical infrastructure operators react in near real-time to emerging threats.

## Key Features (powered by LLMs)
- **Accurate Triage**: Reducing "false positives" in vulnerability matching to prevent unnecessary downtime in OT environments.
- **Automated Summarization**: Distilling complex technical exploits into actionable briefs for plant operators.
- **Protocol Mapping**: Mapping unstructured advisory text to structured formats like CSAF (Common Security Advisory Framework) or VEX.
